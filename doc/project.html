LOD生成

大场景展示需要使用LOD技术。重构场景重建流程，进行LOD重建。可重建任意大的场景。

  1. 开发两种方案： 1)几何+纹理 LOD， 2)纹理 LOD。 两种情况各有不同的用途。

  2. 纹理贴图拼缝优化（ 专利） ： 优化全景图贴图拼缝处存在误差， 两个层次： 刚性和非刚性。 刚性： 使用位姿引导的稀疏匹配优化图像位姿， 使用 Mesh 引导的稠密对应优化图像位姿。 非刚性： 使用 Mesh 引导的稠密对应优化图像像素非刚性形变从而优化贴图。

  3. 大场景的无缝模型构建（ 专利） ： 使用分块合并方案， 分块重建。 然后使用无缝模型合并方案， 合并模型。构建分割合并策略。 构建无缝 Mesh 合并算法。 合并的 Mesh 保持拓扑关系。

  4. 大场景纹理贴图方案： 使用可见性信息优化遮挡情况判定， 使用分块接边信息进行颜色调整

激光点云标定

  1. 单线激光雷达旋转轴标定： 使用点到平面距离为误差项， 优化旋转轴和平面方程， 构建最小二乘， 求解最优旋转轴

  2. 激光与相机标定： 使用图像特征点对应三维点在平面上的信息构建最小二乘优化激光与相机位姿

  3. 使用 Qt 开发可视化的刺点工具， 手工刺点进行标定， 及检查标定精度结果

  4. 激光与相机投影深度遮挡剔除： 相机中心和激光中心存在平移， 造成激光点云投影到图像上存在遮挡情况。 构建数学函数映射， 将激光点云时序信息和旋转角度映射到全景图中， 保持三维空间中的邻域关系到全景图中的二位邻域关系不变， 从而构建二维三角网， 使用三角网进行遮挡剔除

模型拍摄质量评价体系建立

构建经纪人拍摄房屋质量体系， 建立模型质量与拍摄方案的评价关联性。 采用拍摄点位密度， 连通性， 完整性进行评价。

  1. 点位密度： 使用 Occupancy Grid 构建点位拍摄密度指标

  2. 连通性： 构建拍摄点位连通图， 根据连通图分析房屋漫游连接指标， 分析不连通的情况， 使用门的特征进行自动联通。

  3. 完整性： 使用 Occupancy Grid 与 Mesh 空洞识别， 构建拍摄完整性指标

  4. 全局位姿评价： 使用遮挡体积评价指标进行全局位姿评价。

模型优化相关

  1. 基于图像的 Mesh 优化 纹理有更好的细节， 基于纹理信息来优化 Mesh， 使得模型可以更好的表达细节

  2. Poisson 重建 Mesh 优化方案（ 专利） 针对 Poisson 重建 Mesh 在缺点的情况下缺点区域会出现”鼓包”，构建能量优化方程使得 投影平面面积最小的方案来优化该区域。

  3. 室内场景低置信度和不确定点云全局优化 单张全景图深度图是有 AI 生成， 深度图置信度差， 且尺度一致性差， 导致最终模型差。 综合使用图像， 点云及室内平面行特点进行全局位姿优化， 充分利用多视图信息和深度 图置信度， 处理不一致平面， 生成正确， 平整， 光滑的室内模型

全景图相关

  1. 全景图竖直矫正（ 专利） ： 全景图拍摄由于三脚架等不可能完全竖直。 竖直矫正可以简化位姿拼接算法的复杂度。 提取全景图直线段， 根据全景图特点构建 RANSAC， 求解最优竖直轴， 最后根据竖直轴求解最小旋转矩阵， 然后矫正全景图。

  2. 全景图拼接优化 使用直线+半稠密方案来优化全景图拼接。 使用金字塔策略， lm 算法逐层优化全景图拼接

点云拼接相关

  1. 点云粗拼算法（ 专利） 使用 3-congruent Points 算法来进行点云粗拼接。 解决室内点云粗拼接问题。

  2. 室内场景位恣评价算法（ 专利） 传统点云拼接依靠重叠点数量， 然而在室内场景中， 每个点位不能看到全局信息， 仅仅依靠重叠数量并不能够表达拼接结果。 室内场景具有封闭特点， 使用遮挡体积来来描述拼接结果。 为融合多种位姿拼接提供了统一的判定标准

  3. 深度图引导图像特征描述的位姿求解方案（ 专利） 使用深度图信息+图像， 对图像进行局部变换后进行特征描述， 解决大角度下图像匹配差的问题

图像稠密匹配算法

实现基于 CUDA 的 Patch MVS 算法， 实现图像生成稠密点云。 效果， 效率， 占用资源等指标超过所有开源算法。

图像 SfM 算法

实现增量式 SfM 算法， 并优化算法效率， 包括图像特征提取， 匹配， BA 优化。

镶嵌线编辑软件

是一款实时编辑预览镶嵌线结果的软件。 因为镶嵌线设计到的图像可能非常多， 设计并实现超大规模图像浏览算法（ 实时 LOD） 。 实现图像混合 OpenGL 渲染模式， 来支持镶嵌线实时编辑结果展示

DSM 内插算法

SGM 稠密重建
